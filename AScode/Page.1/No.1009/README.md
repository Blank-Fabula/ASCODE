### 문제 설명 

여러가지의 문자들의 입력이 들어올 때 소문자이면은 대문자로 대문자이면은 소문자로 바꿔주는 프로그램을 만들어보자. <br>

### 입력 설명 

첫번째 줄에는 테스트케이스가 들어온다. <br>
그 다음 줄 부터는 문자 1개씩 들어온다. <br>

### 출력 설명 

대문자이면 소문자로, 소문자이면 대문자로 출력을 한다. <br>
