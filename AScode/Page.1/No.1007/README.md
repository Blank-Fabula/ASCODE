### 문제 설명 

정수 3개(x, y, z)를 입력 받아 x, y를 더한 결과를 z로 나눈 나머지를 구하여 출력해 보자. <br>

### 입력 설명 

세개의 정수(x, y ,z)를 한 줄 입력 받는다. <br>
정수의 범위는 int형의 범위를 벗어나지 않는다. <br>

### 출력 설명 

x와 y를 더한 값을 z로 나눈 나머지 <br>
