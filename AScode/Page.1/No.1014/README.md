### 문제 설명 

차차극장 에서 새로운 영화를 개봉하였다. 영화를 개봉한 후 1주일 동안 관람한 관객의 수를 매일 집계 하였다고 하자. 1주일 동안 관람한 관객의 총 인원수를 이용하여 일일 평균 관객의 수를 출력 해보자. <br>

### 입력 설명 

숫자 7개를 입력을 받는다. 각 입력은 양수이다. <br>

### 출력 설명 

1일 평균 관객수는 X명 입니다. <br>
숫자 X는 소수점을 표기 하지 않습니다. <br>
X를 출력하시오. <br>
